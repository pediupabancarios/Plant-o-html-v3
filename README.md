## 📱 Instalar como App no celular
- **iPhone:** Safari → Compartilhar → "Adicionar à Tela de Início"
- **Android:** Chrome → Menu (⋮) → "Adicionar à tela inicial"

## 🔥 Firebase (sincronização em tempo real)
Para ativar trocas, repasses e avisos ao vivo entre os médicos:
1. Crie um projeto em [console.firebase.google.com](https://console.firebase.google.com)
2. Crie um Firestore Database em **Modo de Teste**
3. Substitua `"COLE_AQUI"` no topo do `index.html` com suas credenciais
4. Faça novo commit no GitHub

## 🚀 Como ativar o GitHub Pages
1. Acesse seu repositório no GitHub
2. Vá em **Settings → Pages**
3. Em **Source**, selecione `Deploy from a branch`
4. Branch: **main** / Pasta: **/ (root)**
5. Clique **Save**
6. Aguarde 1-2 minutos — a URL aparece no topo da mesma página
