Configuração do banco de dados compartilhado

⏱ Leva apenas 5 minutos — feito uma única vez

1 Criar conta no Supabase

Acesse supabase.com e clique em "Start your project". É gratuito, não precisa de cartão.

Crie um novo projeto com qualquer nome (ex: medescala), escolha a região South America.

2 Criar a tabela (sem SQL!)

No painel do Supabase, clique em "Table Editor" no menu à esquerda.

Clique em "New table" e configure assim:

Nome da tabela: medescala_data

Desative o "Enable Row Level Security (RLS)" (toggle para OFF).

Adicione 2 colunas clicando em "+ Add column":

Coluna 1: nome = key    tipo = text    (marque PRIMARY KEY)
Coluna 2: nome = value    tipo = text

Clique em "Save". ✅

3 Copiar as credenciais

Vá em Project Settings (ícone de engrenagem ⚙️) → API.

Copie a Project URL e a anon/public key e cole abaixo:

Suas credenciais ficam salvas só no seu dispositivo.
